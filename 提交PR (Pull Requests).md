# 提交PR (Pull Request)

1. fork
2. 克隆到本地
   ```bash
   git clone ${Repository}
   ```
4. 编辑
5. push到自己的github
   ```bash
   git add ./
   git commit -m "comments"
   git push
   ```
6. 提交PR
   在分支选项下有一行`This branch is 1 commit ahead of ${owner}/${Repository}:${branch}.`，后面有个`Contribute`按钮，点击后再点`Open pull request`
